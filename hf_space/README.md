# Chat with Ruby

A witty AI chat assistant, powered by Groq's free hosted Llama 3.3 70B API.

## Setup

1. Get a free API key at https://console.groq.com/keys
2. In this Space's Settings → "Variables and secrets", add a secret named `GROQ_API_KEY` with that key.
3. The Space will rebuild automatically and the chat will go live.
